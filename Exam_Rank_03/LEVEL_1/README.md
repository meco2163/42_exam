# 🇬🇧 English

## Level 1 — Exam Rank 03

Level 1 is the first stage of the exam. This level focuses on basic C programming knowledge and file operations.

### Questions

| Question | Topic | Difficulty | Description |
|:---------|:------|:----------:|:------------|
| [broken_gnl](broken_gnl/) | File I/O, Static Variables | ⭐⭐ | Fix a broken `get_next_line` function |
| [filter](filter/) | stdin/stdout, String Replace | ⭐⭐ | Replace specific words with `*` from stdin input |

### Required Fundamentals

- **File Descriptor (fd)**: Integer identifier used by the OS to track open files
- **Static Variables**: Variables that retain their value between function calls
- **Dynamic Memory**: Memory management with `malloc` / `free`
- **stdin/stdout**: Standard input and output streams

# 🇫🇷 Français

## Level 1 — Exam Rank 03

| Question | Sujet | Difficulté | Description |

## Contributing

Contributions are welcome! If you would like to add resources to this repository, please open a **Pull Request** or an **Issue**.

### What you can contribute

- **New questions** and their solutions that you encountered in exams
- **Alternative approaches** for existing solutions
- **Subject** files
- **Test programs** or test cases
- Useful **additional resources**

### Requirements for contributions

When opening a request, please make sure to include the following information:

1. **The question(s)** you encountered during the exam
2. **Your 42 intra username**
3. **The campus** where you are studying (e.g. 42 Kocaeli, 42 Istanbul, etc.)

> [!IMPORTANT]
> Only questions that correspond to **known exam update dates** will be accepted. This ensures that the repository stays accurate and up to date.

### How to contribute

1. **Fork** this repository
2. Create a new branch (`git checkout -b new-question/question-name`)
3. Commit your changes (`git commit -m 'New question added: question-name'`)
4. Push your branch (`git push origin new-question/question-name`)
5. Open a **Pull Request**

## Exam Tips

- There is **no internet access** during the exam — do not try to memorize solutions without understanding them.
- You will get a random question from Level 1 and a random question from Level 2. You need to solve both.
- You will have access to `man` pages, so you can look up function prototypes even if you don't remember them.
- Your exam duration is **3 hours**.

## Katkıda Bulunma

Katkılarınız memnuniyetle karşılanır! Bu repoya kaynak eklemek istiyorsanız lütfen bir **Pull Request** veya **Issue** açın.

### Neler katkıda bulunabilirsiniz?

- Sınavlarda karşılaştığınız **yeni sorular** ve çözümleri
- Mevcut çözümler için **alternatif yaklaşımlar**
- **Subject** dosyaları
- **Test programları** veya test case'leri
- Faydalı **ek kaynaklar**

### Katkı için gerekli bilgiler

Talep açarken lütfen aşağıdaki bilgileri eklediğinizden emin olun:

1. Sınavda karşılaştığınız **soru(lar)**
2. **42 intra kullanıcı adınız**
3. Eğitim aldığınız **kampüs** (ör. 42 Kocaeli, 42 İstanbul vb.)

> [!IMPORTANT]
> Yalnızca **sınavın güncellendiği bilinen tarihlere** ait sorular kabul edilecektir. Bu, reponun doğru ve güncel kalmasını sağlamak içindir.

### Nasıl katkıda bulunulur?

1. Bu repoyu **fork**'layın
2. Yeni bir branch oluşturun (`git checkout -b yeni-soru/soru-adi`)
3. Değişikliklerinizi commit'leyin (`git commit -m 'Yeni soru eklendi: soru-adi'`)
4. Branch'inizi push'layın (`git push origin yeni-soru/soru-adi`)
5. Bir **Pull Request** açın

## Contribuer

Vos contributions sont les bienvenues ! Si vous souhaitez ajouter des ressources à ce dépôt, veuillez ouvrir une **Pull Request** ou une **Issue**.

### Ce que vous pouvez contribuer

- **Nouvelles questions** et leurs solutions rencontrées lors des examens
- **Approches alternatives** pour les solutions existantes
- Fichiers de **sujets**
- **Programmes de test** ou cas de test
- **Ressources supplémentaires** utiles

### Informations requises pour contribuer

Lors de l'ouverture d'une demande, veuillez vous assurer d'inclure les informations suivantes :

1. **La ou les question(s)** que vous avez rencontrées pendant l'examen
2. **Votre nom d'utilisateur 42 intra**
3. **Le campus** où vous étudiez (ex. 42 Kocaeli, 42 Istanbul, etc.)

> [!IMPORTANT]
> Seules les questions correspondant aux **dates de mise à jour connues de l'examen** seront acceptées. Cela permet de garantir que le dépôt reste précis et à jour.

### Comment contribuer ?

1. **Forkez** ce dépôt
2. Créez une nouvelle branche (`git checkout -b nouvelle-question/nom-question`)
3. Committez vos changements (`git commit -m 'Nouvelle question ajoutée : nom-question'`)
4. Pushez votre branche (`git push origin nouvelle-question/nom-question`)
5. Ouvrez une **Pull Request**
